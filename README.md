Componente Card: Agricultura Regenerativa
Este projeto é uma página web simples que apresenta um card interativo sobre Agricultura Regenerativa. O objetivo é destacar práticas sustentáveis de forma visualmente atraente e dinâmica, utilizando HTML, CSS e JavaScript.

🚀 Funcionalidades
Design Responsivo: Card centralizado e adaptável para diferentes tamanhos de tela.

Interatividade (Toggle): Um botão "Saiba Mais" que exibe ou oculta informações adicionais sobre as práticas principais da agricultura regenerativa.

Tags de Categoria: Identificação visual do tema principal (Sustentabilidade).

🛠️ Tecnologias Utilizadas
HTML5: Estruturação semântica do conteúdo.

CSS3: Estilização e animações (arquivo style.css).

JavaScript: Lógica para alternar a visibilidade das informações extras (arquivo script.js).
